## 🚀 About the Project

StudentFocus is a self-built web application designed to help students manage:

- Semester goals (SGPA, focus area)
- Daily learning schedules
- Weekly goals
- DSA practice tracking
- Project tracking
- Life calendar (365-day consistency view)
- Light / Dark mode
- Authentication & personalized data

Each user has **their own private data**, goals, and schedule.

---

## 🧠 Features

- 🔐 Secure authentication (Supabase)
- 📊 Semester dashboard
- 📅 Daily learning tracker
- 🎯 Weekly goals
- 🧩 Project & DSA tracking
- 🟢 Life calendar (365-day visual tracker)
- 🌙 Light / Dark mode
- 📱 Mobile-friendly UI

---

## 🛠 Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Styling:** Tailwind CSS
- **Backend / Auth:** Supabase
- **State Management:** React hooks
- **Deployment-ready**
