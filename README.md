# StudentFocus 🎯

StudentFocus is a personal productivity and skill-growth web application designed for students to **track goals, improve consistency, and build real-world skills** such as public speaking and group discussion (GD) — all in one place.

This project is built with a mobile-first mindset and is actively evolving as part of my learning and development journey as a Computer Science student.

---

## 🚀 Features

### 📊 Dashboard
- Overview of semester growth
- Quick access to all core modules
- Clean, distraction-free UI

### 🎯 Goals Tracker
- Set academic and personal goals
- Track progress over time
- Helps build consistency and accountability

### 📅 Life Calendar
- Visual representation of long-term planning
- Encourages goal-oriented thinking

### 💻 Projects Section
- Add and manage learning or portfolio projects
- Track project status and technologies used

### 🧠 DSA Tracker
- Track Data Structures & Algorithms practice
- Helps maintain daily problem-solving habit

### 🗣️ Speaking Lab (GD & Public Speaking Practice)
A dedicated module to improve communication skills:

- **Daily Challenge** – Timed speaking on randomly selected topics
- **Free Practice** – Practice speaking without pressure
- **GD Simulator** – Simulates real group discussion scenarios
- **Current Affairs Topics** included
- **History & Stats** – Track total sessions and speaking time
- Visual progress tracking to show improvement over time

This feature is especially useful for:
- Placement GD rounds
- Technical interviews
- Students from non-English medium backgrounds

---

## 📱 Mobile Friendly
- Fully responsive UI
- Optimized for daily use on mobile
- Can be added to home screen and used like an app

---

## 🛠️ Tech Stack

- **Frontend:** React + TypeScript
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **State & Storage:** Local state (backend integration planned)
- **Version Control:** Git & GitHub
- **Deployment:** Render
